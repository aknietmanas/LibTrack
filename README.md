# LibTrack
LibTrack - это приложение на JavaFX, предназначенное для учета книг, авторов, посетителей и операций выдачи. Цель проекта - создать удобный инструмент для библиотекаря, который позволит автоматизировать основные процессы работы библиотеки: хранение данных о фонде книг, управление авторами и читателями, а также фиксирование выдачи и возврата книг.
