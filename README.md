# LibTrack
**LibTrack** - это приложение на **JavaFX**, предназначенное для учета книг, авторов, посетителей и операций выдачи. Цель проекта - создать удобный инструмент для библиотекаря, который позволит автоматизировать основные процессы работы библиотеки: хранение данных о фонде книг, управление авторами и читателями, а также фиксирование выдачи и возврата книг

# Технологический стек
## Frontend / UI:
* **JavaFX** — основная библиотека для построения графического интерфейса
* **FXML** — используется для структурирования окон и сцен
* **CSS** — отвечает за визуальное оформление интерфейса

## Backend / Логика:
* **Java 17+** — язык разработки
* **Java Collections (List, Map, HashMap, ObservableList)** — применяются для хранения и обработки данных внутри приложения
* **MVC/MVVM архитектура** — контроллеры, сервисы и DAO разделяют ответственность и упрощают поддержку проекта

## База данных:
* **MySQL** (или SQLite - в зависимости от выбранного способа развертывания)
* **JDBC** — используется для подключения и выполнения SQL-команд
* _В проект включён SQL-файл для создания всех таблиц_

# Пользовательский интерфейс
Интерфейс создаётся с помощью JavaFX (FXML + CSS) и состоит из нескольких экранов:
**1. Экран входа**
  * ввод логина и пароля
  * проверка пользователя по базе
**2. Главный экран (Dashboard)**
  * навигация по разделам:
    "Книги", "Авторы", "Посетители", "Выдача", "Статистика"
**3. Управление книгами**
  * таблица всех книг
  * поиск и фильтры (по автору, жанру, названию)
  * CRUD: добавить, редактировать, удалить
  * отображение статуса книги (доступна/занята)
**4. Управление авторами**
  * таблица авторов
  * CRUD-операции.
**5. Управление посетителями**
  * список читателей
  * CRUD-операции.
**6. Выдача и возврат книг**
  * выбор книги
  * выбор посетителя
  * регистрация даты выдачи
  * возврат книги
**7. Статистика**
  * диаграммы JavaFX:
  * самые популярные книги
  * количество выданных книг по месяцам
  * активность посетителей
    
_**UI оформлен в современном стиле: ровные панели, аккуратные отступы, светлая или тёмная цветовая схема, читаемые шрифты, единый стиль кнопок и таблиц**_
